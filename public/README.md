Para completar la configuración de SEO, necesitas añadir una imagen `og-image.png` en esta carpeta.

### Especificaciones:
- **Nombre del archivo:** `og-image.png`
- **Dimensiones:** 1200x630 px
- **Formato:** PNG
- **Peso máximo recomendado:** 1MB

### Contenido sugerido:
- Tu nombre: "Linamaría Martínez"
- Tu título: "Desarrolladora Full Stack"
- Tech stack visual (iconos de React, Next.js, TypeScript)
- Colores del brand: #6366f1 (primary), #8b5cf6 (secondary), #ec4899 (accent)

Puedes crear esta imagen usando herramientas como:
- [Figma](https://figma.com)
- [Canva](https://canva.com)
- [OG Image Generator](https://og-image.vercel.app/)

Una vez creada, simplemente coloca el archivo `og-image.png` en esta carpeta `/public/`.
